Enter the full path to the Helix QAC project file to be analyzed.

If the project file is part of the project in the repository, then it will be checked out in the project structure into the Jenkins workspace.

It is also possible to create a Helix QAC project file as a prior build step: that step will determine the location of the project file.

If the project file is inside of the Jenkins workspace, then you can just supply the relative path to it from the project root directory.
