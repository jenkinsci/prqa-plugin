Configure the parameters for uploading to QA·Verify. In each Jenkins job you can select the QA·Verify server to upload to when the Upload results to QA Verify option is enabled.

Click Add or Delete to administer the QA·Servers. Each server entered here will appear in the drop down list in the Select server configuration in each job.

  • Configuration name is the just a symbolic name to refer to this parameter set in the Jenkins Job configuration.
  • Host name is either the IP address or name of the computer that the QA·Verify server is running on.
  • User name and Password is for the account on the QA·Verify server that has permission to upload snapshots. Usually the user is upload and the password is set in the User Management screen in QA·Verify.
  • Upload port number is the QA·Verify port number which accepts uploads. This is configured when setting up the server; the default is 22230.
  • View server port number is the port number of the QA·Verify web interface. The default is 8080.
  • View server protocol is either HTTP or HTPPS depending on whether QA·Verify is accessed via a secure connection.
