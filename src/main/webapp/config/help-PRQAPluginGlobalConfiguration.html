Configure the parameters for uploading to Dashboard Server. In each Jenkins job you can select the Dashboard Server to upload to when the Upload results to Dashboard Server option is enabled.

Click Add or Delete to administer the QA·Servers. Each server entered here will appear in the drop down list in the Select server configuration in each job.

  • Configuration name is the just a symbolic name to refer to this parameter set in the Jenkins Job configuration.
  • Host Address is either the IP address or name of the computer that the Dashboard Server is running on, relative to the Jenkins server.
  • User name and Password is for the account on the Dashboard Server that has permission to upload snapshots. Usually the user is upload and the password is set in the User Management screen in QA·Verify.
  • Upload port number is the Dashboard Server port number which accepts uploads. This is configured when setting up the server; the default is 22230.
  • View server port number is the port number of the Dashboard Server web interface. The default is 8080.
  • View server protocol is either HTTP or HTPPS depending on whether Dashboard is accessed via a secure connection.
  • External Url is the a global link that is exposed on the job page for user to click through to the QA Verify server.
